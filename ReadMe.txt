Visitor Managment System (VMS) :

Is a Spring based web application for managing visitor's entry/exit to/from any company or organization for internal security porpose

VMS global functionalities : 

   - Visitor's visits requests management
   - Visitor's appointments requests management
   - Visitors visits management
   - User management
   
VMS Technical Environment :

- Java EE technology 
- Spring framework 
- JPA/ Hibernate framework
- JSF framework(facelet, Primefaces components API, Bootfaces components API, Bootstrap )
- Spring Security
- Java mail
- Tomcat server
- MYSQL database
- Spring Tool Source (STS IDE)
- MVC, DAO design pattern
